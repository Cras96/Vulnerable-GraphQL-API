# Security Policy

## Intended use

This repository is intentionally vulnerable and exists only for academic
research, controlled demonstrations, and security education.

Run it only on `localhost`, `127.0.0.1`, or an isolated private lab environment.
Do not expose it to the public internet.

## Reporting concerns

If you believe this repository is being misused, or if you identify accidental
exposure of real secrets, personal data, or unsafe deployment instructions,
please contact the repository maintainer or open an issue describing the concern.

Do not include real credentials, tokens, personal data, or exploit output from
third-party systems in public issues.

## Authorization boundary

The included payloads and examples are intended only for this local testbed or
for systems where explicit written authorization has been granted by the system
owner.
