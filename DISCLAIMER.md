# Disclaimer

This project is an intentionally vulnerable GraphQL API created for academic
research, controlled demonstrations, and security education.

The application must only be run locally (`localhost` / `127.0.0.1`) or inside
an isolated private lab environment controlled by the user. It must not be
deployed as a public internet-facing service.

The payloads and examples included in this repository are intended only for this
testbed or for systems where the tester has explicit written authorization from
the system owner. They must not be used against third-party services, production
systems, or public GraphQL endpoints without permission.

All data is synthetic. Do not add real personal data, medical data, production
secrets, API keys, credentials, or tokens to the repository.

This repository is not an exploit framework and is not intended to support
unauthorized access, scanning, denial-of-service activity, data exfiltration, or
any other harmful activity. Users are responsible for complying with applicable
laws, institutional policies, and authorization requirements.

The software is provided without warranty and is not suitable for production
use.
